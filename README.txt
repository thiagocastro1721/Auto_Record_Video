Este script grava automaticamente a tela enquanto um vídeo é executado no Google Chrome, utilizando o OBS em segundo plano.

Ao iniciar, o usuário deve informar a duração da gravação. 

A gravação é finalizada automaticamente quando o tempo definido é atingido. 

A gravação também pode pode ser interrompida a qualquer momento pelo usuário através do atalho Ctrl + Shift + Q.

O OBS deve ser configurado com atalhos de teclado não numéricos 1 para iniciar gravação e 2 para terminar a gravação.

O teclado numérico é obrigatório e ficará dedicado para digitação da duração.

Caso não disponha de um teclado numérico dedicado edite o código e o OBS com outros atalhos de teclado compatíveis com a sua máquina.

Reserve o máximo de memória RAM e processamento para um melhor desempenho.

Desenvolvido para Windows 11.
