Este script grava automaticamente a tela enquanto um vídeo é executado no Google Chrome, utilizando o OBS em segundo plano.

Ao iniciar, o usuário deve informar a duração da gravação em minutos. O processo é finalizado automaticamente quando o tempo definido é atingido ou pode ser interrompido a qualquer momento pelo usuário através do atalho Ctrl + Shift + Q.

Teclado numérico necessário, caso contrário edite o código com outros atalhos de teclado compatíveis com a sua máquina.

Reserve o máximo de memória RAM e processamento para um melhor desempenho.

Desenvolvido para Windows 11.
